# 🏃‍♂️ Manuel Tactique : Phase Offensive — Le Jeu dans les Couloirs Latéraux

Ce document définit les principes d'animation offensive sur les ailes pour étirer les blocs compacts.

## 🧠 1. Analyse Conceptuelle du Jeu de Couloir
Le jeu de couloir consiste à utiliser l'amplitude maximale du terrain pour contourner les blocs axiaux compacts. L'objectif est d'y créer des supériorités numériques ou qualitatives afin d'adresser des centres dangereux dans la surface de réparation.

---

## 📐 2. Principes Tactiques du Jeu de Couloir

### A. Le Dédoublement Extérieur (Overlap)
* **Principe :** Le défenseur latéral passe dans le dos de son ailier pour proposer une solution le long de la ligne.
* **Mise en œuvre :** L'ailier repique vers l'intérieur pour attirer le latéral adverse, libérant le couloir pour la course de son propre latéral.

### B. Le Dédoublement Intérieur (Underlap)
* **Principe :** Le latéral propose une course diagonale vers l'intérieur dans la surface de réparation.
* **Mise en œuvre :** L'ailier conserve la largeur maximale, le latéral plonge dans le demi-espace libéré.

### C. L'Isolement Qualitatif (1 vs 1)
* **Principe :** Mettre l'ailier percutant dans les meilleures conditions pour éliminer son vis-à-vis direct.
* **Mise en œuvre :** Surcharger l'axe pour forcer l'adversaire à se recentrer, puis changer rapidement d'aile vers l'ailier isolé.

### D. La Relation de Triangle (Ailier - Latéral - Milieu)
* **Principe :** Assurer une circulation fluide pour déstabiliser le couloir adverse.
* **Mise en œuvre :** Triangulations constantes pour trouver le troisième homme lancé.

---

## ⏱️ 3. Séance d'Entraînement Type : Animation sur les Ailes

### 📋 Bloc 1 : Échauffement — Triangulations sur l'aile (15 mins)
* **Structure :** Exercice technique de passes en triangle avec dédoublements répétés.

### 🏃 Bloc 2 : Exercice 1 — Situation de 2 vs 1 sur le côté (20 mins)
* **Structure :** Ailier + Latéral contre un défenseur. Trouver la meilleure option pour centrer.

### ⚔️ Bloc 3 : Exercice 2 — Phase Tactique 7 vs 6 avec focus ailes (25 mins)
* **Structure :** Terrain avec deux couloirs latéraux délimités.
* **Règles :** Le ballon doit obligatoirement transiter par un couloir latéral avant d'entrer dans la surface pour marquer.

### 📐 Bloc 4 : Jeu Final — Match standard (20 mins)
* **Règles :** Tout but marqué après un centre consécutif à un dédoublement réussi compte double.

---

## 🗣️ 4. Consignes Clés du Coach sur le Terrain (Coaching Points)
* **"Proposez des courses à haute intensité !"** : Le dédoublement doit se faire à pleine vitesse.
* **"Fixez le latéral !"** : Ailiers, provoquez votre défenseur direct pour libérer l'espace dans son dos.
* **"Changez de rythme !"** : Passez de la préparation lente à l'accélération sur le côté.
